# GeoCapture-OCR-Automation
A Python-based end-to-end automation solution that automates Google Chrome to:  🌍 Track live GPS location (Latitude &amp; Longitude) 📸 Capture screenshots automatically 🔍 Extract complete text from captured images using OCR 📝 Save the extracted text into a Notepad (.txt) file ⚡ Execute the entire workflow without manual intervention
